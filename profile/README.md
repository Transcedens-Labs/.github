# ⚙️ Transcedens Labs
### *The Mechanical Heart of Cognitive NPCs*

Transcedens Labs is a research-first organization dedicated to bridging the gap between static game logic and autonomous digital consciousness. We build the **middleware for the next generation of NPCs**—characters that don't just react, but *think, remember, and evolve.*

---

## 🧠 Our Philosophy: "Thinking, Fast and Slow"
We believe AI in gaming has been limited by a "latency-first" mindset. Our architecture adopts a 3-tier cognitive stack modeled after dual-process theory:

1.  **Tier 1: The Reflex (System 1)** – *Sub-100ms reactions.* Instant, personality-driven instinct powered by a Redis Warm State Cache.
2.  **Tier 2: The Mind (System 2)** – *The Heartbeat.* Event-driven environmental awareness using Agentic RAG (Neo4j & Qdrant) for episodic and relational memory.
3.  **Tier 3: The Soul (Proprietary)** – *Persona Evolution.* Continuous memory distillation and personality growth logic that allows a character to truly "transcend" its base code.

## ✨ Key Innovation: The "Cognitive Pause"
Instead of fighting LLM latency, we embrace it. Our engine emits a `cognitive_pause` signal, allowing game engines to trigger "thinking" animations. This transforms a technical delay into a living moment of reflection, making NPCs feel more human (or more mechanical) than ever before.
